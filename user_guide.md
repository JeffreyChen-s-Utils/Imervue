# Imervue 使用手冊

> 一款輕量、快速的 GPU 加速圖片瀏覽器，支援瀏覽、編輯、標註、批次處理。

---

## 快速開始

打開 Imervue 後，你會看到三個區域：

```
┌──────────┬──────────────────────┬──────────┐
│  資料夾  │                      │  EXIF    │
│  樹狀圖  │     圖片瀏覽區       │  資訊欄  │
│          │                      │          │
└──────────┴──────────────────────┴──────────┘
```

- **左邊**：資料夾目錄，點選資料夾就能瀏覽裡面的圖片
- **中間**：圖片顯示區，會以縮圖網格展示所有圖片
- **右邊**：EXIF 資訊欄，顯示圖片的拍攝資訊

---

## 開啟圖片

| 方法 | 操作 |
|------|------|
| 開啟資料夾 | `檔案` → `開啟資料夾`，選擇你要瀏覽的目錄 |
| 開啟單張圖片 | `檔案` → `開啟圖片`，選擇一張圖片 |
| 拖曳開啟 | 直接把圖片或資料夾拖進視窗 |
| 從檔案總管開啟 | 右鍵圖片 → `Open with Imervue`（需先註冊檔案關聯） |
| 最近開啟 | `檔案` → `最近開啟`，快速回到之前看過的資料夾 |

**支援的圖片格式：**
PNG、JPEG、BMP、TIFF、WebP、GIF、APNG、SVG、RAW（CR2、NEF、ARW、DNG、RAF、ORF）

## 瀏覽圖片

### 縮圖網格模式

開啟資料夾後，所有圖片會以縮圖網格排列。

| 操作 | 方法 |
|------|------|
| 瀏覽圖片 | 滑鼠滾輪上下捲動 |
| 移動畫面 | 按住滑鼠中鍵拖曳 |
| 進入大圖模式 | 左鍵點擊任一張縮圖 |
| 調整縮圖大小 | 上方選單 `縮圖大小` → 選擇 128 / 256 / 512 / 1024 |
| 調整縮圖密度 | `縮圖大小` → `縮圖排列密度` → 緊湊 / 標準 / 寬鬆 |
| 框選多張圖片 | 左鍵拖曳框選 |
| 方向鍵移動 | `↑` `↓` `←` `→` 平移畫面，按住 `Shift` 可微調 |
| Hover 預覽彈窗 | 滑鼠懸停縮圖 500 ms 顯示放大預覽 |

縮圖左邊緣會顯示色彩標籤色條；左上角 ❤ = 最愛、右上角 ★ = 書籤、左下角顯示評分星。

### 清單（詳細）模式

按 `Ctrl+L` 切換至可排序的清單檢視，欄位為：預覽／標籤／名稱／解析度／大小／類型／修改時間。雙擊列或按 `Enter` 可開啟大圖模式，再按 `Esc` 回到清單。

### 大圖模式（Deep Zoom）

點擊縮圖後進入大圖模式，可以高畫質瀏覽單張圖片。

| 操作 | 方法 |
|------|------|
| 放大／縮小 | 滑鼠滾輪、或觸控板捏合 |
| 拖曳平移 | 按住滑鼠中鍵 |
| 上一張圖 | `←` 左方向鍵、或觸控板向左滑 |
| 下一張圖 | `→` 右方向鍵、或觸控板向右滑 |
| 跨資料夾 | `Ctrl+Shift+←` / `Ctrl+Shift+→` 跳到前／下一個有圖片的兄弟資料夾 |
| 瀏覽歷史 | `Alt+←` 返回、`Alt+→` 前進（類似瀏覽器） |
| 依編號跳轉 | `Ctrl+G` 開啟跳頁對話框 |
| 隨機圖片 | `X` 隨機跳一張 |
| 適應寬度 | 按 `W` |
| 適應高度 | 按 `Shift + W` |
| 重設縮放 | 按 `Home` |
| 回到縮圖模式 | 按 `Esc` |
| 全螢幕 | 按 `F`，再按一次退出 |
| 劇場模式 | `Shift+Tab` 隱藏所有選單 / 狀態列 / 樹狀圖 |
| 顯示疊加資訊 (OSD) | `F8` 切換檔名/尺寸/大小疊加層；`Ctrl+F8` 顯示 Debug HUD |
| 像素檢視 | `Shift+P`，放大至 ≥ 400% 即顯示像素網格與游標下 RGB/HEX |
| 色彩模式 | `Shift+M` 循環：正常 → 灰階 → 反相 → 懷舊 |

## 整理圖片

### 評分與最愛

在大圖模式下，可以快速幫圖片打分數：

| 操作 | 按鍵 |
|------|------|
| 加入最愛 ❤ | `0` |
| 評 1～5 星 | `1` `2` `3` `4` `5`（再按一次取消） |

### 色彩標籤 (F1–F5)

獨立於星等之外的 Lightroom 風格顏色 flag，適合快速分類（例如：紅=刪除候補、綠=精選、藍=待修）。

| 操作 | 按鍵 |
|------|------|
| 紅 / 黃 / 綠 / 藍 / 紫 | `F1` / `F2` / `F3` / `F4` / `F5`（再按一次清除） |
| 縮圖多選後批次套用 | 框選多張 → 按對應 F 鍵 |
| 依色彩篩選 | `篩選` → `依色彩標籤` → 選擇顏色 / 任一色 / 無標籤 |

狀態列右側會顯示當前圖片的色彩 chip；縮圖左邊緣會有對應色條。

### 書籤

把常用的圖片加入書籤，方便日後快速找到。

| 操作 | 方法 |
|------|------|
| 加入書籤 | 大圖模式下按 `B` |
| 管理書籤 | `檔案` → `書籤` |

### 標籤與相簿

用標籤和相簿分類管理你的圖片。

| 操作 | 方法 |
|------|------|
| 開啟管理介面 | 按 `T` 或 `檔案` → `標籤與相簿` |
| 幫圖片加標籤 | 右鍵圖片 → `加入標籤` |
| 加入相簿 | 右鍵圖片 → `加入相簿` |
| 依標籤篩選（單選） | `篩選` → `依標籤` |
| 依相簿篩選（單選） | `篩選` → `依相簿` |
| **多標籤過濾 (AND/OR)** | `篩選` → `多標籤過濾…` — 勾選多個標籤/相簿，選擇「任一 OR」或「全部 AND」 |

### 排序與篩選

| 功能 | 選單位置 |
|------|----------|
| 依名稱排序 | `排序` → `依名稱` |
| 依修改日期排序 | `排序` → `依修改日期` |
| 依檔案大小排序 | `排序` → `依檔案大小` |
| 依解析度排序 | `排序` → `依解析度` |
| 升冪／降冪 | `排序` → `升冪` / `降冪` |
| 依副檔名篩選 | `篩選` → `JPEG` / `PNG` / `RAW` 等 |
| 依評分篩選 | `篩選` → `依評分` |
| 依色彩標籤篩選 | `篩選` → `依色彩標籤` |
| **進階過濾** | `篩選` → `進階過濾…` — 解析度範圍 / 檔案大小範圍 / 方向（橫/直/正方）/ 修改日期區間 |
| 清除篩選 | `篩選` → `清除篩選` |

## 編輯圖片（修改分頁）

切換到上方的 **「修改」** 分頁，就能進入編輯模式。

```
┌────────┬──────────────────────┬────────────┐
│  工具  │                      │  繪圖屬性  │
│  列    │   畫布（直接繪圖）    │  調色、筆刷 │
│        │                      │  曝光調整  │
└────────┴──────────────────────┴────────────┘
```

也可以在大圖模式下按 `E` 或右鍵 → `修改` 進入。

### 標註工具（左邊）

| 工具 | 圖示 | 說明 |
|------|------|------|
| 選取 | ⬚ | 選取已畫的標註，可拖曳移動 |
| 裁剪 | ✂ | 裁剪圖片，支援比例預設（自由 / 1:1 / 4:3 / 3:2 / 16:9 / 9:16） |
| 矩形 | ▢ | 畫矩形框 |
| 橢圓 | ◯ | 畫橢圓或圓形 |
| 直線 | ╱ | 畫直線 |
| 箭頭 | → | 畫箭頭 |
| 手繪 | ✎ | 自由手繪線條 |
| 文字 | T | 在圖上加文字 |
| 馬賽克 | ▦ | 框選區域打馬賽克 |
| 模糊 | ◌ | 框選區域加模糊 |

> 在修改分頁中按 `←` `→` 可以切換上一張／下一張圖片。

### 筆刷類型（右邊）

| 筆刷 | 效果 |
|------|------|
| 鋼筆 ✒ | 標準細線，最常用 |
| 麥克筆 🖊 | 較粗、半透明的筆觸 |
| 鉛筆 ✏ | 細淡的線條 |
| 螢光筆 🖍 | 寬大、高透明度，像真的螢光筆 |
| 噴霧 💨 | 噴點效果 |
| 書法 🖋 | 粗細隨筆劃方向變化 |
| 水彩 🎨 | 柔和暈染效果 |
| 炭筆 ▪ | 粗糙質感筆觸 |
| 蠟筆 🖍 | 蠟質手感 |

### 繪圖屬性（右邊）

| 屬性 | 說明 |
|------|------|
| 顏色 | 點擊色塊選擇繪圖顏色 |
| 線寬 | 拖動滑桿調整筆觸粗細（1～40） |
| 不透明度 | 調整透明度（0%～100%） |
| 字型 | 選擇文字工具使用的字型 |
| 字型大小 | 調整文字大小（6～200 px） |

### 影像調整（右邊下方）

| 滑桿 | 功能 |
|------|------|
| 曝光 | 調整整體亮度 |
| 亮度 | 微調明暗 |
| 對比 | 調整明暗對比 |
| 飽和度 | 調整色彩鮮豔程度 |

這些調整是**非破壞性**的，隨時可以按 `重設` 恢復原狀。

### 儲存與復原

| 按鈕 | 說明 |
|------|------|
| 儲存 | 將標註和調整寫入原始檔案 |
| 復原 | 取消上一步操作 |
| 重做 | 重做已復原的操作 |
| 重設 | 清除所有影像調整 |

## 匯出圖片

### 單張匯出

右鍵圖片 → `匯出 / 另存為`

- 選擇格式：PNG、JPEG、WebP、BMP、TIFF
- 調整品質（有損格式可調）
- 預覽檔案大小
- 選擇儲存位置

### 批次匯出

選取多張圖片後，右鍵 → `批次匯出`

- 統一格式轉換
- 設定最大寬度／高度（自動等比縮放）
- 品質控制
- 進度條即時顯示

### 製作 GIF / 影片

選取多張圖片後，右鍵 → `建立 GIF / 影片`

- 支援 GIF 和 MP4 格式
- 可拖曳排列順序
- 設定每秒幀數（FPS）
- 自訂尺寸
- 循環播放選項

## 圖片比較

在縮圖模式下框選 2～4 張圖片，右鍵 → `比較圖片`。

對話框有三個頁籤：

| 頁籤 | 功能 |
|------|------|
| **並排** | 2 張或 4 張同時顯示，各自自適應縮放 |
| **重疊** | 兩張圖疊在一起，拖曳 α 滑桿（A ↔ B）混合顯示 |
| **差異** | 計算 \|A − B\| 每像素差異；增益滑桿（0.1×–20×）可放大細微差異 |

尺寸不同的兩張圖會自動以 Lanczos 對齊 A 的尺寸。超大圖會自動 downscale（長邊 ≤ 2048）以保持即時反應。

### 分割檢視與雙頁閱讀

不用開 Compare Dialog 也能並排兩張：

| 操作 | 按鍵 |
|------|------|
| 分割檢視（任意兩張並排） | `Shift+S` |
| 雙頁閱讀（當前 + 下一張） | `Shift+D` |
| 雙頁閱讀（右至左，漫畫用） | `Ctrl+Shift+D` |
| 返回先前模式 | `Esc` |

### 多螢幕視窗

按 `Ctrl+Shift+M` 會在副螢幕開啟一個無邊框的鏡像視窗，顯示當前大圖模式圖片。主視窗可以繼續瀏覽其他圖片（展覽、雙螢幕工作流很實用）。再按一次 `Ctrl+Shift+M` 關閉。

## 幻燈片

按 `S` 或右鍵 → `幻燈片`，開始自動播放所有圖片。

- 可調整每張停留時間
- 可開啟淡入淡出效果

---

## 搜尋圖片

按 `Ctrl + F` 或 `/`，輸入關鍵字即可搜尋當前資料夾中的圖片名稱。

搜尋支援**模糊匹配**（前綴 > 子字串 > 子序列三級排名）與**子字串高亮**。結果即時更新，雙擊結果或按 `Enter` 就能直接跳到那張圖。

> 若想依**編號**直接跳圖，改按 `Ctrl + G` 開啟跳頁對話框。

## 複製與貼上

| 操作 | 方法 |
|------|------|
| 複製圖片到剪貼簿 | 大圖模式下按 `Ctrl + C` |
| 貼上剪貼簿圖片 | `檔案` → `從剪貼簿貼上`，或按 `Ctrl + V` |

> 自動監控功能開啟後，每當剪貼簿出現新圖片（例如用截圖工具），就會自動開啟標註編輯。

大圖模式下右鍵 → `設為桌布`，一鍵將當前圖片設為系統桌布。

## Windows 檔案關聯

讓你在檔案總管中直接用 Imervue 開啟圖片：

2. 需要系統管理員權限
3. 之後右鍵任意圖片就能看到 `Open with Imervue` 選項

## 額外功能

在上方選單列的 **「額外功能」** 或右鍵選單的 **「額外功能」** 子選單中，可以找到以下工具：

### 圖片淨化重繪

從原始像素重新繪製圖片，徹底移除所有隱藏資料（EXIF、中繼資料、隱寫內容、尾部位元組），並以日期 + 隨機字串重新命名。

| 設定 | 說明 |
|------|------|
| 來源資料夾 | 選擇要處理的圖片資料夾 |
| 包含子資料夾 | 是否遞迴掃描子目錄 |
| 輸出資料夾 | 淨化後的圖片儲存位置 |
| 輸出格式 | 與來源相同、或轉換為 PNG / JPEG / WebP / BMP / TIFF |
| 隨機字串長度 | 檔名中隨機部分的長度（4～32） |
| JPEG 品質 | 輸出為 JPEG 時的壓縮品質（1～100） |

#### AI 放大（可選）

可以將小於目標解析度的圖片使用 Real-ESRGAN AI 放大至常用解析度，保持比例。

| 目標 | 長邊像素 |
|------|----------|
| Full HD | 1920 px |
| 2K | 2560 px |
| 4K UHD | 3840 px |
| 5K | 5120 px |
| 8K UHD | 7680 px |

系統會自動選擇 x2 或 x4 模型，先 AI 放大再以高品質 LANCZOS 縮放至精確目標尺寸。

### 批次格式轉換

將資料夾中的圖片統一轉換為指定格式，支援品質控制。

### AI 圖片放大

使用 Real-ESRGAN AI 模型進行超解析度放大（x2 一般、x4 一般、x4 動漫），支援 CUDA / DirectML / CPU。

### 重複圖片偵測

使用精確雜湊或感知比對（漢明距離）找出重複圖片，可批次移至垃圾桶。

### 圖片整理工具

依日期、解析度、檔案類型、檔案大小或固定數量，將圖片自動分類到子資料夾。

### EXIF 批次清除

移除資料夾中所有圖片的 EXIF、GPS 及其他中繼資料，保護上傳前的隱私。

## 進階功能（v2 新增）

### 命令面板（Ctrl+Shift+P）

用鍵盤快速啟動任意選單動作。按下後輸入關鍵字即可模糊搜尋並執行，不需翻選單。

### Session 儲存與還原

- `檔案` → `Session` → `儲存 Session`：把目前的資料夾、選擇、縮放、檢視狀態存為一個可命名的 session。
- `檔案` → `Session` → `載入 Session`：之後可隨時還原，適合多工作集切換。

### 巨集錄製與重播

1. `額外功能` → `巨集管理員…` 開啟巨集對話框。
2. 按「開始錄製」後進行評分、色彩標籤、最愛、標籤等操作，系統會自動收集每一步。
3. 停止錄製後可命名並儲存，之後選取多張圖片即可一鍵批次重播。
4. `Alt + M` 直接重播最近一次執行過的巨集。

### RAW + JPEG 堆疊

當同一張照片同時存在 RAW 與 JPEG（同檔名、不同副檔名）時，`篩選` → `堆疊 RAW+JPEG 對` 勾選後會合併顯示為單一項目，預設以 RAW 為主檔。

### 外部編輯器整合

1. `檔案` → `外部編輯器設定…` 註冊編輯器（名稱、執行檔路徑、選用引數；用 `{path}` 佔位符代表圖片路徑）。
2. 之後 `檔案` → `以外部編輯器開啟` → 選擇編輯器，即可用當前圖片啟動。

### 聯絡表（Contact Sheet）PDF 匯出

`額外功能` → `聯絡表 PDF…`。可指定列/行、頁面大小（A4/Letter/…）、是否印出檔名標題。適合快速列印整個資料夾的縮圖頁。

### Web Gallery HTML 匯出

`額外功能` → `Web Gallery…`。產生自含式 HTML（內建 Lightbox 燈箱），可選擇是否複製原始圖片到輸出目錄。整個資料夾可直接壓縮後上傳。

### 幻燈片影片（MP4）匯出

`額外功能` → `幻燈片影片（MP4）…`。可設定解析度、FPS、每張停留秒數、淡入淡出秒數。輸出需要 `imageio-ffmpeg` 套件提供 ffmpeg 後端。
